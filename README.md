# 🚀 LÓGIKA Lexer

Un analizador léxico moderno y potente para el lenguaje de programación experimental **LÓGIKA**. Construido con React, Vite y Tailwind CSS, ofreciendo una experiencia de usuario premium con estética Dark Mode y Glassmorphism.

## ✨ Características

- **Análisis en Tiempo Real:** Procesa código LÓGIKA instantáneamente.
- **Detección de Tokens:** Clasificación automática de identificadores, palabras reservadas, números, operadores, strings y comentarios.
- **Gestión Inteligente de Errores:** Resalta fallos léxicos y proporciona una **Sugerencia de Corrección** automática (ej. cierre de comillas o nombres de variables inválidos).
- **Ejemplos Integrados:** Selector de código pre-cargado con casos de éxito y casos de error para pruebas rápidas.
- **Interfaz Premium:** Diseño oscuro profesional con efectos de cristal (Glassmorphism), gradientes y micro-animaciones.

## 🛠️ Tecnologías

- **React 18** - Desarrollo de componentes reactivos.
- **Vite** - Build tool de última generación para un desarrollo fluido.
- **Tailwind CSS** - Framework de utilidades para un diseño robusto y responsivo.
- **JavaScript Regex** - Implementación personalizada de autómatas para el análisis de léxico.

## 💻 Instalación y Uso local

Sigue estos pasos para ejecutar el proyecto en otra computadora:

### Requisitos Previos

- Tener instalado **Node.js** (Versión 16.0 o superior).
- Un gestor de paquetes preferido (**npm**, **yarn** o **pnpm**).

### Pasos para el despliegue

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/ChutinRop/logika-lexer-app.git
   ```

2. **Acceder al directorio:**
   ```bash
   cd logika-lexer-app
   ```

3. **Instalar las dependencias:**
   ```bash
   npm install
   ```

4. **Lanzar la aplicación:**
   ```bash
   npm run dev
   ```

5. **Acceso:**
   Abre tu navegador en la dirección `http://localhost:5173` (o la que indique la terminal).
